ALL TYPES OF BORDER STYLES IN HTML

boders specify what kind of borders it was:

  • dotted-defines a dotted border

  • dashed-defines a dashed border

  • solid-defines a solid border

  • double-defines a double border

  • groove-defines a 3D groved border.the effect depends on the border-color value

  • ridge-defines a 3d riged border.the effect depends on the border-color value

  • inset-defines a 3d inset border.the effect depends on the border-color value

  • outset-defines a 3d outset border.the effect depends on the5f border-color value

  • none-defines no border

  • hidden-defines a hidden border
